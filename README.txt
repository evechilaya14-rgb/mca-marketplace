How to run:
1. Open this folder in VS Code.
2. Open Terminal.
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000

New features added:

- Filter products by category.
- Product categories on submit and available items pages.
- Admin dashboard statistics.
- Admin edit product button.
- Admin delete product button.
- Product image uploads.
- Sample approved products with pictures already available.
- Admin login and logout.

Database:
This version uses database.json, so it does not need SQLite or Python.
